# InfoVis Assignment 2 – Flight Route Projections

This project is a small React + TypeScript + Vite app that visualizes different map projections of flight routes.
It uses D3 and Tissot’s indicatrices for distortion visualization.

## How to Run

```bash
yarn install # Install dependencies
yarn dev     # Start the development server
```

Then open your localhost URL in a web browser to view the app.
